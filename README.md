### Tauri PC 应用

一个基于 Rust Tauri 的 PC 应用, 添加了仪表盘, 图表, 三维场景, TCP数据交换. 可以切换暗黑模式和正常模式.

> TCP 服务连接成功会3秒钟发送一个随机单词.
